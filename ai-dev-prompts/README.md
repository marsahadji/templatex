# Bibliothèque de Prompts d'Ingénierie

Cette bibliothèque contient une collection de prompts modulaires conçus pour standardiser le développement avec la stack Next.js 16, MongoDB et Tailwind CSS.

## Structure du Dépôt

Les prompts sont organisés par catégories fonctionnelles :

### 00-Architecture-Global
- Vision macro et choix technologiques immuables
- Définition stricte de la stack technique
- Règles de structure de dossiers

### 01-Configuration
- Gestion des variables d'environnement
- Configuration de Tailwind et du Design System
- Règles ESLint et Prettier
- Configuration de Next.js

### 02-Database-Data
- Modélisation Mongoose
- Peuplement de données
- Transformation de données ouvertes
- Pipelines d'agrégation MongoDB

### 03-Security-Auth
- Flux d'authentification NextAuth
- Contrôle d'accès basé sur les rôles (RBAC)
- Validation des entrées
- Headers de sécurité

### 04-Middleware-Proxy
- Middleware Next.js
- Limitation de débit API
- Gestion des erreurs

### 05-Backend-Logic
- Actions serveur
- Gestionnaires de route API
- Intégration de services tiers

### 06-Frontend-UI
- Composants serveur
- Composants client
- Hooks personnalisés

### 99-Utils-DevOps
- Standards de commit Git
- Configuration Docker Compose

## Utilisation

Pour utiliser cette bibliothèque de prompts :
1. Identifiez la catégorie pertinente pour votre besoin
2. Consultez le(s) fichier(s) correspondant(s)
3. Adaptez le contenu du prompt à votre contexte spécifique
4. Utilisez le prompt avec votre outil d'IA préféré

Chaque fichier contient un frontmatter YAML avec des métadonnées et une section "Contenu du Prompt" prête à être complétée.
